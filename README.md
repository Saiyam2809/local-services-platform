# Local Services Booking Platform

A full-stack marketplace connecting customers with local service providers.

## Features

### Customer
- Browse service categories
- View providers
- Book services
- Track booking status
- Submit reviews

### Provider
- Manage profile
- Accept / reject bookings
- Update job status
- Toggle availability

### Admin
- Approve providers
- Manage service categories
- Moderate reviews

## Tech Stack

Frontend
- Next.js
- React
- CSS

Backend
- Next.js API Routes
- Prisma ORM

Database
- PostgreSQL

Authentication
- NextAuth
